## Overview

Built a flexible hero banner with short and tall variants and configurable text placement (on or below the banner). Key content types use a fixed layout with limited text positioning to ensure consistency.

## Usage in pages

The website includes two types of standard pages: a Basic Page, which is a regular informational page, and a Section Page, which can contain multiple related pages and has its own dedicated menu. Both page types allow customization of short or tall hero banners, with the option to position them inside or below the page header.
