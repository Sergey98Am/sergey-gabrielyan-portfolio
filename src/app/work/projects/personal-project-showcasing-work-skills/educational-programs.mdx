## Drupal Technical Details

### Relationships

Each Program is linked to exactly one Faculty and, optionally, to a Department through two Entity Reference fields—one for Faculty and one for Department—on the Educational Program Group. This structure enforces a strict hierarchy and ensures that no Program exists without a parent Faculty.

### Form

Department options are filtered based on the selected Faculty in the form.
