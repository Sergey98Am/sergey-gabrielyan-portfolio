### Media configuration

#### Media Types

Media entities have types. A custom media type called Image Widget Crops is defined with an image widget that supports crops.

#### Media View Modes

Media also has view modes similar to content types. View modes are configured for the media type and responsive image styles are assigned to them.

#### Usage in Content View Modes

In the content (or group) type view mode, the media field uses the Rendered entity formatter and the configured media view mode, which applies the responsive image style.
