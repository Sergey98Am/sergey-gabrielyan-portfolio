title: "Multilingual Menu System"
---

Implemented a flexible multilingual navigation system using a **separate-menu structure** for each language (English & Armenian). This approach provides full control over menu content and hierarchy per language, without relying on translation settings or forced synchronization.

**What Implemented:**

- **Dedicated Menus Per Language:**
  - Main Navigation (English)
  - Main Navigation (Armenian)
  - Secondary Navigation (English)
  - Secondary Navigation (Armenian)

- **Improved UX for Native Users:**
